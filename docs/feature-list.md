# 查单后台功能列表

## 1. 账号与权限
- 内部账号登录 / SSO 扩展位
- 角色管理：管理员、客服
- 会话管理：Token/Session 失效时间 8 小时
- 操作日志：登录、刷新、状态变更

## 2. 单号查询
- 单号搜索框（单条查询）
- 批量导入组件（CSV/Excel，格式校验）
- 查询结果表格：状态、更新时间、承运商、备注、责任人
- 轨迹详情弹窗/页签

## 3. 异常处理
- 异常标签管理（待处理、处理中、已解决）
- 备注记录（富文本/Markdown 简写）
- 责任人指派
- 异常列表过滤、排序、导出

## 4. 刷新控制
- 单号手动刷新按钮（调用刷新 API，展示任务状态）
- 自动刷新策略配置：频率、重试次数、任务启停
- Redis 刷新锁防重

## 5. 系统配置
- 17-track API Key 管理（密文存储）
- MySQL/Redis 连接健康检查
- 阈值告警配置（缓存命中率、API 失败率）
